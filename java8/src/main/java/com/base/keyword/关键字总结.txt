final
    变量: 常量值不能被更变，引用值则不能被更变
    类: 这个类不能被继承
    方法: 锁定方法，不能被继承，性能提升

static
    变量、成员方法：被类中所有对象共享，存放在java内存方法区
        方法区用于存储被虚拟机加载的方法信息、常量、静态变量、即时编译后的代码数据，被称为永久代存在
    代码块;只执行一次，静态块-》非静态块-》构造
        静态内部类 ： 不会被加载到内存中，只有在调用的时候才会被加载，有延时的好处，保证线程的安全
        静态代码块与非静态代码块：静态代码块只在第一次new执行，非静态在每每一次都执行
        静态方法与非静态方法：静态方法属类本身，非静态方法属类中生成对象， 静态方法占空间更少
        非静态代码块与构造函数:非静态代码块是对类中所有对象进行初始化，构造函数是对构造的初始化
    导包


this
    引用于当前实例
    调用父类构造要放在第一行
super
    从子类访问父类方法、变量

    this、super 调用父类构造要放在第一行
    不能在static 中， this、super指向的是对象，static 指向的是类
