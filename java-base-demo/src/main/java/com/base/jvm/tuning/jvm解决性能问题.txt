4、双亲委派机制
	工作流程：当一个类加载器收到请求时，自已先不会加载而把请求发送给父类，
		当顶级父类加载不到，子加载器才会尝试自己加载
	意义：
		1、系统类防止内存中出现多份同样的字节码
		2、保证java程序安全稳定运行
		


		
了解 主要 解决性能问题

				及时编译后的代码数量
			类信息

		栈
