queue
	优点：
		1、通过异步处理提高性能（销峰，减少响应时间）
		2、降低系统的耦合
	消息队列使用发布订阅模式
	JMS
		概要:java的消息服务，JMS客户端通过JMS服务异步消息传输
		消息模型：
			点到点：使用队列做为载体，一条消息只能补一个消费者使用
			发布订阅：使用主题做为载体，一条消息能被多个订阅主题的消费者使用
			
	AMQP
		高级消息队列协议（二进制应用层协议）
		RabbitMQ就是基于AMQP协议实现的
		
		activeMQ 性能差
		
		RabbitMQ吞吐量稍逊于 kafka 和RocketMQ ，并发能力好，性能好，廷时低
		kafka 超高的吞吐量，ms级廷迟，劣势消息可能被重复消费
			  大数据领域的实时计算、日志采集等场景，用 Kafka 是业内标准的
